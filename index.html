Commands
All commands
General
!gd user <username> Look up a player's stats.
!gd level <name> Look up a level's info.
!gd plat <name> Look up a platformer level's info.
!gd top Show the top 10 players by stars.
!gd help Show the help message.
Daily / Weekly / Event
!gd daily Show the current daily level.
!gd weekly Show the current weekly demon.
!gd event Show the current event level.
